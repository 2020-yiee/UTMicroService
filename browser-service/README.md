# Browser service

## Development

```bash
npm install && npm run start:dev
```

or

```bash
yarn && yarn start:dev
```

## Test API
1. Import `browser-service.postman_collection.json` to Postman
4. Feel free to test API & remember to update `browser-service.postman_collection.json`

## Requires

- NodeJS
- Postman (optional)
- Yarn (optional)
